Install Evolve

Installing Evolve

This is the one place to start. Installing Evolve takes about 30 minutes of hands-on time once you have your accounts and keys ready — most of the elapsed time before that is gathering API keys and waiting for account approvals.

Install Evolve on macOS

End-to-end setup on a dedicated Mac — a Mac mini or a retired Apple-Silicon MacBook. This is the simplest way to run Evolve.

Install Evolve on Linux (Ubuntu / VPS)

Install the Evolve app on your phone or desktop

The Evolve admin dashboard is a Progressive Web App (PWA) — you can install it as a real app with its own icon, on a phone or on a desktop, straight from the browser. No app store, no separate download.
